# 📰 AMPKHABAR

A modern News Portal Management System built using PHP, MySQL and Bootstrap.

## 🚀 Features

- Admin Login
- News Management (Add, Edit, Delete)
- Category Management
- Breaking News
- Featured News
- Advertisement Management
- Contact System
- Website Settings
- Responsive Design

## 🛠️ Technologies Used

- PHP
- MySQL
- Bootstrap 5
- HTML5
- CSS3
- JavaScript

## 📂 Installation

1. Copy project into xampp/htdocs
2. Import ampkhabar.sql into phpMyAdmin
3. Start Apache & MySQL
4. Open:

http://localhost/ampkhabar
